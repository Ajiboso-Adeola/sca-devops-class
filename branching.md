## Task 2<br>
> Create a new branch named "dev", create a file named "branching.md" with the branch name, commit and push to GitHub

Step by step on how to do this
- Open your terminal

- create a new branch using the command ```git checkout -b dev```

- create a new file ```branching.md```, use the command `touch branching.md`

- git add <branching.md>
- git commit -m "add descriptive message"
- git push
